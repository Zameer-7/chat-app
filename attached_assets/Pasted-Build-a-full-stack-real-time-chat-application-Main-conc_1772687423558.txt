Build a full-stack real-time chat application.

Main concept:
Users can create chat rooms and invite others by sharing a room ID. Anyone with the room ID can join the room and chat in real time.

Tech stack requirements:

Frontend:
Next.js (React + TypeScript)
Tailwind CSS for styling

Backend:
Python FastAPI

Realtime communication:
WebSockets (native FastAPI WebSockets, not Socket.io)

Database:
PostgreSQL using SQLAlchemy ORM

Application features:

1. Home Page
- Button to create a new room
- Input to join an existing room using Room ID
- Input for username

2. Room creation
- Generate a unique room ID automatically
- Display the room ID and shareable link

3. Join room
- User enters room ID and username
- If the room exists they enter the chat

4. Chat system
- Real-time messaging using WebSockets
- Each room should have its own WebSocket channel

Example endpoint:
/ws/room/{room_id}

5. Message persistence
Every message must be stored in the database with:
- id
- room_id
- username
- message
- timestamp

6. Load chat history
When a user joins a room, fetch previous messages from the database and display them.

7. Chat UI
Create a modern WhatsApp-style interface including:
- chat message bubbles
- scrollable chat area
- timestamps
- message input bar
- send button
- responsive design
- dark mode support

8. Project structure

backend/
    main.py
    database.py
    models.py
    websocket_manager.py
    routers/

frontend/
    pages/
    components/
        ChatWindow.tsx
        MessageBubble.tsx
        JoinRoom.tsx
    utils/

9. API endpoints

POST /create-room
GET /messages/{room_id}

10. Provide instructions to run locally:

Backend:
- create virtual environment
- install dependencies
- run FastAPI server with uvicorn

Frontend:
- npm install
- npm run dev

Generate clean, well-structured code with comments explaining the WebSocket room logic and database interaction.
